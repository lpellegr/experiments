## Usage

If you want to import the module with Intellij, you need to run the next command first:

```bash
$ ./gradlew idea
```

You can compile classes as follows:

```bash
$ ./gradlew build
```

Micro-benchmarks are runnable with the following command:

```bash
$ ./gradlew jmh
```
